## УСЛОВНЫЕ КОДЫ (15 кодов)

Каждый условный код может быть добавлен к инструкциям с суффиксом `{cond}`:

| Суффикс | Условие | Флаги | Описание |
|---------|---------|-------|----------|
| EQ | Equal | Z=1 | Равно |
| NE | Not Equal | Z=0 | Не равно |
| CS/HS | Carry Set/Higher or Same | C=1 | Больше или равно (без знака) |
| CC/LO | Carry Clear/Lower | C=0 | Меньше (без знака) |
| MI | Minus | N=1 | Отрицательное |
| PL | Plus | N=0 | Положительное или ноль |
| VS | Overflow Set | V=1 | Переполнение |
| VC | Overflow Clear | V=0 | Нет переполнения |
| HI | Higher | C=1 AND Z=0 | Больше (без знака) |
| LS | Lower or Same | C=0 OR Z=1 | Меньше или равно (без знака) |
| GE | Greater or Equal | N=V | Больше или равно (со знаком) |
| LT | Less Than | N!=V | Меньше (со знаком) |
| GT | Greater Than | Z=0 AND N=V | Больше (со знаком) |
| LE | Less or Equal | Z=1 OR N!=V | Меньше или равно (со знаком) |
| AL | Always | - | Всегда (по умолчанию) |

## ФОРМАТ OPERAND2

Второй операнд (Operand2) может быть:

1. **Константа** (#imm):
   - 8-битное значение, сдвинутое на любое число бит
   - Формы: 0x00XY00XY, 0xXY00XY00, 0xXYXYXYXY

2. **Регистр с опциональным сдвигом** (Rm {, shift}):
   - ASR #n - арифметический сдвиг вправо (1-32)
   - LSL #n - логический сдвиг влево (1-31)
   - LSR #n - логический сдвиг вправо (1-32)
   - ROR #n - вращение вправо (1-31)
   - RRX - вращение вправо с расширением (1 бит через C)

## РЕГИСТРЫ CORTEX-M3

### Регистры общего назначения (R0-R12)
- R0-R7: "Low registers" (доступны во всех инструкциях)
- R8-R12: "High registers" (ограниченный доступ в Thumb-1)

### Специальные регистры
| Регистр | Описание |
|---------|----------|
| R13 (SP) | Указатель стека (MSP/PSP) |
| R14 (LR) | Регистр связи (адрес возврата) |
| R15 (PC) | Счётчик команд |
| xPSR | Регистр состояния программы |
| PRIMASK | Маска прерываний |
| FAULTMASK | Маска ошибок |
| BASEPRI | Базовый приоритет |
| CONTROL | Контроль привилегий и стека |

## ФЛАГИ APSR

| Флаг | Описание |
|------|----------|

## РЕКОМЕНДАЦИИ ПО РЕАЛИЗАЦИИ

### Приоритет реализации (минимальное подмножество):

1. **Базовый набор (для загрузки программы):**
   - MOV, LDR, STR, ADD, SUB, CMP, B, BL, BX, PUSH, POP

2. **Арифметика:**
   - ADD, SUB, MUL, DIV, CMP, CMN

3. **Логика:**
   - AND, ORR, EOR, BIC, TST, TEQ

4. **Ветвления:**
   - B, BL, BX, BLX, CBZ, CBNZ

5. **Память:**
   - LDR, LDRB, LDRH, STR, STRB, STRH, LDM, STM

6. **Системные:**
   - NOP, WFI, SVC, MRS, MSR
