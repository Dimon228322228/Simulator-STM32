## 4. Критические проблемы

### 🔴 Критично (блокирует сдачу)

1. **Прерывания не работают**
   - Нет stacking/unstacking контекста
   - Нет EXC_RETURN механизма
   - Нет связи NVIC с ядром

2. **TIM6 не функционирует**
   - Регистры есть, счёт не идёт
   - Нет генерации прерываний

3. **Нет приёма UART (RX)**
   - Только передача (TX)
   - Нет функции внешнего ввода

4. **Нет отчёта PDF**
   - Требуется по ТЗ (раздел 9)

---

1. **Счётчик тактов упрощён**
   - Все инструкции = 1 цикл
   - Нет дифференциации по PM0056

2. **OpenTelemetry отсутствует**
   - Нет метрик в Go-оркестраторе
   - Нет трейсинга

3. **Системных тестов нет**
   - Нет end-to-end тестов
   - Нет сравнения с реальным железом
