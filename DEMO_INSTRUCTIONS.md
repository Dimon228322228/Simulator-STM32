# 🎬 STM32 Simulator — Демонстрация для преподавателя

## Быстрый старт (WSL / Linux)

### 1. Перейдите в директорию проекта

```bash
cd /path/to/Simulator-STM32
```

### 2. Запустите интерактивное демо

```bash
./run_demo.sh
```

Откроется меню с 6 демо-программами. Выберите номер или нажмите `a` для запуска всех.

### 3. Или запустите конкретное демо вручную

```bash
# Сгенерировать все .bin файлы
python3 demo_programs.py all

# Запустить конкретное демо
./build/stm32_sim arithmetic.bin --max-steps 20
./build/stm32_sim fibonacci.bin --max-steps 50
./build/stm32_sim max_value.bin --max-steps 20
```

### 4. Запустить все тесты

```bash
./run_tests.sh
```

## Для Windows (PowerShell)

```powershell
# Сгенерировать демо
python3 demo_programs.py all

# Запустить интерактивное меню
.\run_demo.ps1

# Или вручную
.\build\stm32_sim.exe arithmetic.bin --max-steps 20

# Запустить тесты
.\run_tests.ps1
```

## Описание демо-программ

| # | Демо | Что показывает | Ожидаемый результат |
|---|------|----------------|---------------------|
| 1 | **Arithmetic** | ADD, SUB, CMP | R0=10, R1=3, R2=13, R3=7 |
| 2 | **Loop Counter** | Цикл 0→5 с BLE | R0=6 (вышел из цикла) |
| 3 | **Fibonacci** | 8 итераций Фибоначчи | R0=21, R1=34 |
| 4 | **Memory (LDR)** | Загрузка констант из Flash | R0=0xDEAD, R1=0xBEEF, R2=0x19D9C |
| 5 | **LED Blink** | XOR-переключение в цикле | R2=1 (5 переключений) |
| 6 | **Max Value** | BGT — ветвление по условию | R2=42 (максимум из 42 и 17) |

---

## Что происходит при запуске

1. **Сборка** — CMake компилирует симулятор и все тесты
2. **Генерация** — Python создаёт бинарные файлы с Thumb-16 инструкциями
3. **Загрузка** — симулятор читает `.bin` во Flash-память (0x08000000)
4. **Выполнение** — цикл fetch-decode-execute для каждой инструкции
5. **Результат** — вывод состояния регистров и статистики
